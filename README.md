Introduction
------------

The project is to show how algorithm works using graphic animation.

Algorithms
------------

This application currently shows the following algorithms: 

**Bubble Sort**

**Merge Sort**
 
**Quick Sort**

**Depth-first Search**

**Breath-first Search**

**Dijkstra's Search**

**A\* Search**

**Binary Search Tree**
